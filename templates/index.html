How the Model Works

Input features are label-encoded using encoders fitted during training to ensure identical mapping.

Customer remarks are cleaned and vectorized via TF-IDF with top 200 bigram features.

Numerical features (response time, item price) are log-transformed then StandardScaled.

A 3-layer ANN tuned via RandomizedSearchCV outputs CSAT probabilities for scores 1-5.
